# Smart Attendance System Showcase Website

This is a clean, modern, and responsive website designed to showcase the "Smart Attendance System Using Face Recognition" project.

## Project Structure

- `index.html`: The main HTML file containing the website structure and content.
- `style.css`: The CSS file for styling, including responsive design and animations.
- `script.js`: The JavaScript file for interactivity (smooth scrolling, mobile menu, scroll animations).

## How to Run

1.  **Open the Folder**: Navigate to the folder where you saved these files.
2.  **Launch the Website**: Double-click on `index.html` to open it in your default web browser.

## Features

-   **Responsive Design**: Works on mobile, tablet, and desktop.
-   **Modern UI**: Clean layout with gradients, shadows, and Poppins font.
-   **Animations**: Smooth fade-in effects on scroll.
-   **Interactive Elements**: Hover effects, smooth scrolling navigation, and a mobile menu.

## Customization

-   **Images**: Replace the placeholder text in the "Showcase / Demo" section with actual screenshots of your project.
-   **Links**: Update the "Explore Features" button or social links as needed.
